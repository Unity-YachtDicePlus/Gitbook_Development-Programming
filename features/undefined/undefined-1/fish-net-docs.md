## Fish-Net 설치

1. 유니티 [에셋 스토어](https://assetstore.unity.com/packages/tools/network/fishnet-networking-evolved-207815)에 접속하여 Fishnet을 유니티 프로젝트에 설치
2. [Github](https://github.com/ooonush/FishyUnityTransport)에 접속하여 유니티 프로젝트에 해당 [링크](https://github.com/ooonush/FishyUnityTransport.git?path=Assets/FishNet/Plugins/FishyUnityTransport)를 다운 <img src="../../../.gitbook/assets/image (4).png" alt="" data-size="line">

<div align="left"><figure><img src="../../../.gitbook/assets/image (1) (1).png" alt="" width="375"><figcaption></figcaption></figure></div>



현재 설치된 목록

* Multiplayer Center
* Multiplayer Quickstart Content
* Multiplayer Play Mode
* Multiplayer Services
* FishyUnityTransport
* FishNet: Networking Evolved





## 멀티플레이 테스트

1. 모든 설치가 완료되었다면 NetworkManager 프리펩을 하이어라키에 추가\
   ![](../../../.gitbook/assets/image.png)
2. 플레이어 프리펩을 하나 만들기(Network Object 컴포넌트가 포함되어 있는)\
   ![](<../../../.gitbook/assets/image (1).png>)
3. NetworkManager > PlayerSpawner 컴포넌트에서 Player Prefab 추가, Spawns에 스폰지역 추가\
   ![](<../../../.gitbook/assets/image (2).png>)



4. Window > Multiplayer > Multiplayer Play Mode에서 테스트 플레이어 추가\
   ![](<../../../.gitbook/assets/image (3).png>)
